The Poorman’s OS X

Nov 17th, 2003 | Comments

This past weekend, I happened about an Apple store in my neighborhood. I finally got to test drive OS X a little bit – yeah, yeah, I’m a little behind ok. Anyway, as someone who learned how to use a computer on the Mac back in the early 90’s, I don’t really have a desire to go back. I’m not saying Windows is more superior nor is Linux better, but for me, aside from the Mac’s look and feel, I don’t see why Mac users practically have orgasms when a new release of OS X comes out. Come on. To me, Macs are over-priced. Plus, there’s just too much stuff that doesn’t run on a Mac that I absolutely can’t do my work without. If I want a Unix style system, I can have one for free… Linux. Which brings me to the topic of this post. The image above is the latest desktop on my home Linux machine. I installed Ximian’s new desktop which is a modified version of Gnome and applied an Aqua-like theme to it. This just goes to show that you don’t have to shell out beaucoup bucks to get a Mac. Just take one of your Windows machines and install a Linux distro on it, and you’re off.
