The Best Source for Bookmarklets

Jun 17th, 2003 | Comments

This site has the most complete source of bookmarklets I’ve ever run across. My favorite one is the edit styles bookmarklet which brings up a window that allows you to change a site’s stylesheets on the fly and have the changes instantaneously be reflected on the page… this only works on Mozilla Firebird though… but still very cool.
